KittyWonderland
===============

Each achievement has its own folder.
Before running any instruction, please move to the according folder.

## How to compile

Run:
```bash
make
```

## How to compile and execute

Run:
```bash
make exec
```

## How to run the test suite

Run:
```bash
make test
```
